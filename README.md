## 前言

2018年5月1日北大学生发布了《五一|不应被遗忘的群体——2018年北大校园工人访谈全纪录》，但不久后即被删除。随后发生了工友被约谈、部分工友被公安机关要求停止和学生接触、部分前往走访的学生被北大保卫部和警察威胁等一系列事件。
